# Evade-v0.5-Alpha-WIP-

Evade is a multiplayer fighting game made in Scratch. The project focuses on creating a complete online experience with features that are uncommon on the platform, including multiplayer gameplay, player progression, cosmetics, achievements, and a custom soundtrack.

Development is still ongoing, so the game is constantly changing as features are added, removed, or reworked. This repository contains the project's source code, assets, and other files used during development.

## Features

* Multiplayer gameplay
* Character progression
* Unlockable cosmetics
* Achievement system
* Original soundtrack
* Custom visual effects
* Performance optimizations
* Regular updates

## Project Goals

The project also serves as a place to experiment with optimization techniques, networking, game systems, and animation.

Everything in this repository is a work in progress, and some systems may be incomplete or change over time.
